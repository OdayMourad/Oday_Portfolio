Project 4: Natural Language Processing with Disaster Tweets

Dataset: more than 10,000 real tweets. Target: Predict which tweets are talking about disaster and which are not. Project Overview: Discovering the data and applying Exploratory Data Analysis to extract some patterns related to if a tweet is about disaster or not. Building deep neural network model using Lstm to start predicting.

Project 2: Titanic Survived Prediction
